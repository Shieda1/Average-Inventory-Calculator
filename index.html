What is the average inventory?
The average inventory is a measurement of the number of a specific product or set of products over two or more time periods. The average inventory is the mean of inventory during a specific period of time.
How to calculate average inventory?
To calculate average inventory you should use the following formula:
average inventory = (current inventory + previous inventory) / number of periods
Example: company x's inventory for september, october and november is $416,000, $256,000 and $534,000, calculate the average inventory?
using the average inventory formula the answer will be like this:
average inventory = (current inventory + previous inventory) / number of periods
average inventory = ($416,000 + $256,000 + $534,000) / 3
average inventory = $402,000
Why use average inventory?
Companies use the average inventory to find out how fast they can turn inventory into sales. It is also used to give an overview for a specific period of time.
